1.用到的语句

JavaScript中的函数是一段可重复使用的代码块，具有特定的功能。函数可以接收参数、执行一系列的操作，并返回一个结果。

以下是JavaScript中函数的常见用法和功能：

1. 定义函数：使用关键字 function 来定义一个函数，可以指定函数名、参数列表和函数体。

2. 调用函数：通过函数名加上括号 () 来调用函数，并可以传递参数供函数使用。

3. 参数传递：函数可以在定义时指定参数列表，当函数被调用时，可以传递实际的参数值给形式参数，供函数内部使用。

4. 返回值：函数可以使用关键字 return 来返回一个值，返回值可以在函数调用处被捕获和使用。

5. 匿名函数：可以创建没有函数名的匿名函数，通常作为另一个函数的参数或赋值给变量使用。

6. 高阶函数：JavaScript中函数也是一种数据类型，可以将函数作为参数传递给其他函数，或者将函数作为返回值返回。

7. 闭包：函数内部可以访问外部函数的变量和参数，形成闭包。这使得函数可以访问其创建时的环境变量，即使外部函数已经执行完毕。

8. 函数方法：函数也是对象，因此可以具有自己的属性和方法。例如，可以使用 Function.prototype 对函数进行扩展和修改。

9. 递归：函数可以直接或间接地调用自身，这称为递归。递归函数可以解决一些需要重复操作的问题。

10. 回调函数：在异步操作中，可以将一个函数作为参数传递给另一个函数，并在适当的时候调用该函数，这被称为回调函数。

    函数在JavaScript中具有非常重要的地位，可以用于模块化代码、封装功能、提高代码复用性和可维护性等。通过合理使用函数，可以使代码更加清晰、简洁和易于理解。

2.关于标签的id和标签的类名

1.document.getElementById()
    JavaScript 中 Document 对象的一个方法，用于通过元素的唯一标识符（ID）获取对应的元素对象
    在 HTML 文档中，我们可以为元素指定一个唯一的 ID 属性，以便于在 JavaScript 中使用该 ID 来操作和获取该元素。document.getElementById() 方法接受一个参数，即要查找的元素的 ID，然后返回一个表示该元素的对象。
以下是 document.getElementById() 方法的详细解释：
    document：document 对象代表整个 HTML 文档，是 JavaScript 访问和操作 HTML 文档的入口点。
    getElementById()：这是 document 对象的一个方法，用于根据元素的 ID 获取元素对象。
    参数：作为 getElementById() 方法的参数，我们传入一个字符串，即要查找的元素的 ID。这个字符串需要与 HTML 文档中的某个元素的 ID 属性值完全匹配。如果找到匹配的元素，则返回对应的元素对象；如果没有找到匹配的元素，则返回 null。
    返回值：getElementById() 方法的返回值是一个表示找到的元素的对象。我们可以使用这个对象来进行各种操作，比如修改元素的样式、内容或属性，添加事件监听器等。
2.document.getElementsByClassName(className) 方法是 JavaScript 中 Document 对象的一个方法，用于根据元素的 class 类名获取对应的元素对象。
    该方法接受一个参数 className，表示查找的元素的类名。它返回一个 HTMLCollection 对象，其中包含了所有具有指定类名的元素。HTMLCollection 对象类似于数组，可以使用下标访问其中的元素。
以下是 document.getElementsByClassName() 方法的详细解释：
    document：document 对象代表整个 HTML 文档，是 JavaScript 访问和操作 HTML 文档的入口点。
    getElementsByClassName()：这是 document 对象的一个方法，用于根据元素的类名获取元素对象。
    参数：作为 getElementsByClassName() 方法的参数，我们传入一个字符串，即要查找的元素的类名。这个字符串需要与 HTML 文档中的某个元素的 class 属性值完全匹配。如果找到匹配的元素，则以 HTMLCollection 返回所有匹配的元素；如果没有找到匹配的元素，则返回空的 HTMLCollection 对象。
    返回值：getElementsByClassName() 方法返回的是一个 HTMLCollection 对象，其中包含了所有具有指定类名的元素。HTMLCollection 对象类似于数组，可以使用下标访问其中的元素。
